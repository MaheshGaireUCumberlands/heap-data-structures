# Assignment 4 — Heaps and Priority Queues

Contents:
- `src/heapsort.py` — Heapsort implementation
- `src/priority_queue.py` — `Task` dataclass and `BinaryMaxHeap` implementation
- `src/scheduler.py` — simple scheduler simulation using the heap
- `benchmarks/compare_sorts.py` — empirical comparisons of sorting algorithms
- `report.md` — analysis, complexity proofs, and discussion

Quick start:

Run the heapsort demo:
```bash
python3 src/heapsort.py
```

Run the scheduler demo:
```bash
python3 src/scheduler.py
```

Run sorting benchmarks:
```bash
python3 benchmarks/compare_sorts.py
```

See `report.md` for analysis and instructions to extend benchmarks.
